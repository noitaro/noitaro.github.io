Rakuten Mini に Magisk を 入れて Root化 する

2020-06-13
Uncategorized
812 words
  • android
  • magisk
  • rakuten-mini
  • rooted

前回、Rakuten Mini の ブートローダーをアンロックしました。

https://noitaro.github.io/rakuten-mini-bootloader-unlock/

今回は ルート化 です。

準備

ブートローダーが解除してある事が前提です。まだの方は 過去記事 をどうぞ。

Magisk適応済みブートイメージを手に入れる

まず、Magiskパッチが当たった boot_magisk_patched.img を手に入れます。

「AndroPlus」こちらの一番下にある ファームウェアダンプ から、自分の端末に合ったファイルをダウンロードします。

https://androplus.org/Entry/14664/

ソフトウェアバージョンを確認

端末情報の ソフトウェアバージョン と同じ boot_magisk_patched.img を入れないと、ブートループに陥って Android が起動しなくなります。

私の場合は「C330AE_9.0_RMN_JP_65_SS」だったので n1ckn4m3さんの boot_magisk_patched.img をダウンロードしました。

Magisk適応済みブートイメージ を焼く

では boot_magisk_patched.img もダウンロード出来たので、早速焼いちゃいましょう。

Bootloader モード

PC と Rakuten Mini を USBケーブルで接続し、下記コマンドを実行して 端末を Bootloader モードにします。これは前回と同じ流れですね。

Rakuten Mini 端末側はこんな画面になります。

bootイメージ書き込み

端末が Bootloaderモード になったら、下記コマンドを実行して boot_magisk_patched.img を書き込みます。

追伸

いきなり「fastboot flash boot」は危険なのでやめましょう。最初に「fastboot boot」をして問題なく起動できることを確認してから「fastboot flash boot」をしましょう。

Magisk を 入れて Root化 する

この段階で Root化出来る状態になったので、最後の仕上げ Magisk Manager を 入れて Root化 します。

ここから先は Rakuten mini 端末で作業します。

Magisk Manager をダウンロード

Chrome で「Magisk Manager」を開きます。

https://magiskmanager.com/

Menu の Download を押し「Download Magisk Manager [1.85 MB]」をダウンロードします。

Magisk Manager をインストール

Root化

Magisk Manager を起動すると Root化 が始まります。指示に従って操作し再起動したら Root化完了です。

こんな感じになればOK

おわりに

次は、カスタムリカバリの TWRP を入れてみますかー

最後に、充電用じゃなくてデータ通信用の USB3.0ケーブル じゃないと adbコマンド や fastbootコマンド が使えないので持ってない人は買っておくといいですよ。この USB3.0ケーブル なら、太くて丈夫だし急速充電にも対応しているからおすすめです。でもウサギには噛みちぎられるけどね。
