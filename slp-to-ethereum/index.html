SLP を Ethereum に変換する

2021-11-14
Uncategorized
1.1k words
  • axie-infinity

Axie Infinity (アクシーインフィニティ) のアドベンチャークエストをすると SLP (SMOOTH LOVE POTION) がもらえます。

毎日こつこつアドベンチャークエストをこなしある程度たまったので、SLP を Ethereum に変換したいと思います。

ちなみに1日最大75SLP もらえるのですが、私はPVPで勝てたことがないので1日50SLPが限界でした。毎日続けていればそれなりに貯まるので、悔しくなんかないですよ。

Binance取引所のアカウント作成

SLP から Ethereum に変換するには Binance取引所 を使います。

アクシーを買った時に使った MetaMask より手数料が安いためです。ちょっと前に対応したらしい。

アカウント作成は、指示に従って進めて行けば問題ないでしょう。

SLP を移動させるためには本人認証も必要なので、そこまで完了させておきます。

本人認証に10日前後かかると思ってたんですが、1日で終わりました。

SLP を Ronin Wallet に移動

では早速 Axie Infinity の My account ページ を開いて SLP を Ronin Wallet に移動しましょう。

Claim Tokens を選らんで「Claim SLP」を選択します。

とりあえず今回は 850SLP を移動させます。

Ronin Wallet にログインしてから「Claim SLP now」を押すと、

Ronin Wallet の確認画面が表示されるので「Confirm」を押します。

少し時間が掛かりますが、Ronin Wallet に移動することができます。

前回移動させたのと合わせて 2203SLP になりました。

SLP を Ronin Wallet から Binance に移動

Ronin Wallet に持ってきた SLP を Binance に移動します。

Binance の「フィアットと現物」を選択し、

コイン検索で「SLP」を検索して、SLPの入金を選択します。

入金のネットワークは「Ronin」を選択します。

初回は Ronin の入金アドレスを持っていないので「アドレス取得」を選択して Binanceに入金するためのアドレスをもらいます。

SLP入金用アドレスが発行されるので、発行されたアドレスに SLP を移動させます。

Ronin Wallet に戻って「Send」から SLP を Binance に移動させます。

BinanceのSLP入金アドレスをコピペで貼り付けて、ASSETをSLPに変えて、MAXボタンを押して全てのSLPを移動させます。

「Confirm」ボタンを押してしばらく待つと、BinanceにSLPが入金されます。

SLP を Bitcoin や Ethereum に変換

SLPから直接 Bitcoin にも変換できるみたいです。調べてたら初めて知った。

コンバート

コンバートを使うと簡単に SLP を Bitcoin や Ethereum に変換することができます。

振替元を SLP にして、振替先を Bitcoin や Ethereum すれば簡単に変換することができます。

現物取引

現物取引で SLP を 米ドル (USD) や Ethereum にすることもできます。

「SLP」で検索して「SLP/USDT」や「SLP/ETH」を選んで取引をします。

おわりに

遊ぶだけでお金がもらえるのは魅力的だと思いました。時間があればの話ですが。

私はPVPが苦手なので、ゲームが上手い人なら 毎月2250SLP貯めて日本円で18,000円相当は稼げるんじゃないでしょうか。

1アカウントだとお小遣い程度ですが、複数アカウント作って遊べばそれなりになりますね。

あと、初回にアクシーを3匹買ったのでその資金を回収するまでは続けたいと思います。

私が買ったアクシーは一番安い子ですが、毎日クエストするだけなので問題ないです。PVPはムリ。

最初の3匹は 1匹 0.04 ETH の子を買いました。3匹で 0.12 ETH、当時のレートで約40,000円。

思ってたより早く回収できそうなのでホントよかったです。ETHが値上がりしてるから。

余裕があれば アクシーをブリードして、生まれた子を競売にかけて稼ぐってこともやってみたいです。

そこまでハマる自信はないですが。
